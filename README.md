## Getting Started

### Requirements

1. NetBeans IDE 8
2. JDK 8
3. JRE 8

### Installation

1. Extract file ```anime-master.zip``` in your NetBeans projects folder.
2. Rename folder ```anime-master``` to ```anime```.

### Usage

1. Open NetBeans IDE.
2. Open project ```anime```.
3. Run project.
